# Git Commands Reference

## Setup & Config

### git --version
Checks installed Git version.

Example:
```bash
git --version
```

### git config --global user.name
Sets Git username.

Example:
```bash
git config --global user.name "Jaishree Chaure"
```

### git config --global user.email
Sets Git email.

Example:
```bash
git config --global user.email "your-email@gmail.com"
```

---

## Basic Workflow

### git init
Creates a Git repository.

Example:
```bash
git init
```

### git add .
Stages all files.

Example:
```bash
git add .
```

### git commit -m
Creates a commit.

Example:
```bash
git commit -m "Initial commit"
```

---

## Viewing Changes

### git status
Shows repository status.

Example:
```bash
git status
```

### git log
Shows commit history.

Example:
```bash
git log
```

### git branch

Shows branches.

Example:

git branch

### git checkout

Switch branches.

Example:

git checkout feature

### git merge

Merges branches.

Example:

git merge feature
